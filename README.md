# StoneMind - 9x9围棋对弈

一个专为**9x9围棋**设计的浏览器对弈界面，后端由DeepSeek大模型驱动。  
9路围棋节奏更快、战斗更激烈，适合快速对局和移动设备体验！  

---

## ✨ 功能特性  

- � **专注9x9**：专门为9路围棋优化，棋局更紧凑、节奏更快。  
- 🎮 **智能对弈**：与DeepSeek AI进行围棋对弈，支持选择黑白子。  
- ⚡ **快速对局**：9x9棋盘平均15-30分钟完成一局，适合碎片时间。  
- 📱 **移动优先**：专为手机和平板优化，支持触摸和鼠标双重操作。  
- 🔄 **智能确认**：移动设备采用两步确认，桌面设备支持悬停预览。  
- 🥢 **提子统计**：自动显示被吃掉的棋子数量。  
- 🎭 **生动头像**：艾莎 vs 机器人，轮到谁时头像会放大发光。  
- 🧠 **专业AI**：针对9路围棋特点优化的AI提示词，战术更精准。  
- 🔧 **API测试**：内置DeepSeek API Key有效性测试功能。

## 🎯 为什么选择9x9围棋？

**传统19x19围棋 vs 9x9围棋对比：**

| 特点 | 19x19 | 9x9 |
|------|--------|-----|
| 对局时长 | 1-3小时 | 15-30分钟 ⭐ |
| 学习难度 | 复杂 | 易上手 ⭐ |
| 移动体验 | 格子较小 | 格子更大 ⭐ |
| 战斗频率 | 较少 | 频繁激烈 ⭐ |
| 适用场景 | 专业比赛 | 休闲娱乐 ⭐ |

**9x9围棋的优势：**
- 📱 **移动友好**：更大的格子，更适合触摸操作
- ⚡ **快节奏**：每手棋都很关键，容错率低，更刺激
- 🎯 **易学习**：局面相对简单，适合围棋初学者
- 🏃 **快速上手**：无需长时间投入，随时开始一局

## 📱 移动端优化  

### 触控交互
- 使用 **Pointer Events API** + **Touch Events** 双重支持  
- **双重确认机制**：第一次点击预览，第二次点击确认落子  
- **防误触优化**：`touch-action: none` 防止页面滚动干扰  
- **格子放大**：9x9棋盘格子更大，最适合手指操作  

### 设备适配  
- **响应式布局**：自动适配横屏和竖屏模式  
- **屏幕适配**：动态调整棋盘大小适应不同屏幕  
- **高DPI支持**：自动适配高分辨率屏幕的设备像素比  
- **触摸反馈**：半透明预览石子 + 绿色确认圆圈  

### 桌面增强
- **鼠标悬停**：实时显示淡色预览石子  
- **直接点击**：鼠标模式下直接点击落子，无需确认  
- **精确定位**：像素级精确的坐标计算
